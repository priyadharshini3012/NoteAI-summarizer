# NoteAI - AI Text Summarizer

## 🚀 Features
- Text summarization
- Clean UI
- Fast processing

## 🛠 Tech Stack
- Frontend: React
- Backend: Python (Flask / FastAPI)
- AI: NLP model

## 📦 Installation

### Backend
pip install -r requirements.txt
python app.py

### Frontend
npm install
npm start
